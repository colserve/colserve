# ColServe

Late-interaction visual document retrieval in 5 lines of Python.

Open-source library and hosted service for indexing and searching PDFs, scanned documents, and visually-rich content using Col-models (ColPali, ColQwen2, ColSmol, ColIdefics3) with multi-vector late-interaction retrieval.

**Status:** placeholder — full library shipping in 8 weeks.

Sign up for launch notifications at [colserve.dev](https://colserve.dev).

## What's in this repo right now

- `python/` — placeholder PyPI package (`pip install colserve`)

Both packages currently print a launch message on import. The real library lands in this repo over the next 8 weeks.
